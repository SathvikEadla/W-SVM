Ctor, can call with no arguments (uses default arguments for construciton). All space is on the stack. Object will exist but is not valid until some fitting fucntion is called

Parameters
scores_to_dropis this object for prediction, if so how many top scores to drop when fitting
fitting_sizetail size for fitting. With small data the defaults are fine.. if you have millions make it larger for better predictions
verboseis the code chatty on errors during fitting, useful for debugging
alphaband for confidence interfals
translate_amountshifting data to ensure all is positive.. if data is very broad and you want some probabilities for all points you can make it larger..
